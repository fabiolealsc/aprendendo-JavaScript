Java Script/HTML/CSS

Java Script :

O JS é uma linguagem de programação, cuja fução é fazer as interações, os calculos e lógica de sua aplicaçõa WEB.

HTML:

O HTML(Hyper Text Markup Language) é uma linguagem responsável por colocar as informaçoa em sua aplicaçao como textos, títulos, imagens, vídeos e etc...

CSS:

O CSS (Cascading Style Sheets) tem o papel de dar estilo a sua aplicação, como cor dos textos, tamanhos dos títulos e distribuição das imagens na tela.
